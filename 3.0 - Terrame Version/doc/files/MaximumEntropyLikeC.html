MaximumEntropyLikeC

Component developed based on MaxEnt behaviour, considering the attributes values of the samples cells to calculate the potencial of the use on the whole cellular space.

Arguments

  • potentialData: A table with the data need to run the component.
  • potentialData.attributesClass: A list of the select attributes (categorical) to analysis.
  • potentialData.attributesPerc: A list of the select attributes (range) to analysis.
  • potentialData.cellUsePercentage: A percentage to consider a cell a sample.

Return value:

    The modified component.

Usage

P1 = MaximumEntropyLikeC
{
 potentialData =
 {
   -- Region 1
   {
     -- floresta
     {
       cellUsePercentage = 75,

         "assentamentos",
         "rodovias",

       }
