NeighSimpleRule

Simple component developed as teaching material. Not to be used in real applications. Estimates cell potential for a given use according to the percentage of cells of the same type in a Moore neighbourhood.

Arguments

  • run: Handles with the execution method of a NeighSimpleRule component.
  • verify: Handles with the verify method of a NeighSimpleRule component.

Return value:

    The modified component.
