LuccME version 3.0 implementation.

Types

AllocationBySimpleOrdering Simple model developed as teaching material.
AllocationClueLike Based on the process of competition among classes in the same cell, adjusted iteratively to reach the demand when all cells are considered, as described in Verburg et al.
AllocationClueLikeSaturation Modification of the AllocationClueLike component.
AllocationClueSLike Based on the process of competition among classes in the same cell, adjusted iteratively to reach the demand when all cells are considered, as described in Verburg et al.
AllocationClueSNeighborOrdering Based on the process of competition among classes in the same cell, adjusted iteratively to reach the demand when all cells are considered, as described in Verburg et al.
AttractRepulseLogisticRegression Modification of the LogisticRegression combining cellular automata based models ideas.
ComputeInputThreeDateMaps A component (demand algorithm), where the demand inputs set by the user on the configuration file is used to distribute on the cells the correct among of this demand according to the parameters rules.
ComputeInputTwoDateMaps A component (demand algorithm), where the demand inputs set by the user on the configuration file is used to distribute on the cells the correct among of this demand according to the parameters rules.
InverseDistanceRule Simple component developed as teaching material.
LinearRegression Uses linear regression techniques to estimate the cell potential for change.
LogisticRegression Uses logistic regression techniques to compute cell probability for each land use.
LuccMEModel Handles with a LuccME model behaviour.
MaximumEntropyLikeC Component developed based on MaxEnt behaviour, considering the attributes values of the samples cells to calculate the potencial of the use on the whole cellular space.
MaximumEntropyLikeD Component developed based on MaxEnt behaviour, considering the attributes values of the samples cells to calculate the potencial of the use on the whole cellular space.
NeighAttractionLogisticRegression Modification of the LogisticRegression combining cellular automata based models ideas.
NeighInverseDistanceRule Simple component developed as teaching material.
NeighSimpleRule Simple component developed as teaching material.
PreComputedValuesINPE A component (demand algorithm), where the demand inputs set by the user on the configuration file is used to distribute on the cells the correct among of this demand according to the parameters rules.
SpatialLagLinearRoads Modification of the SpatialLagRegression component to allow the use of simple linear regression in specific cases, when roads are created or paved.
SpatialLagRegression Similar to the LinearRegression approach, but relies spatial regression techniques to estimate the regression cover (considers the spatial dependence of the land use).
