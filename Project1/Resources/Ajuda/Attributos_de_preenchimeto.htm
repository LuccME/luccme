Nesta seção o usuário deve informar quais são os atributos que serão usados para preencher as células.

 

                                               Figura 1: Attributos de Preenchimeto

 

Para adicionar os arquivos que contém os dados, o usuário possui duas opções:

1. Clicar no Botão +, localizado na parte inferior esquerda da janela Atributos de Preenchimento:

                             Figura 2: Botão Adicionar Attributos

 

Após clicar no botão a janela de seleção de aquivos aparecerá:  

                                         Figura 3: Janela Seleção de Atributos

 

Os formatos de arquivos suportados atualmente na interface são: .shp e .tif.

A opção de multipla seleção está habilitada nesta janela, não sendo necessário escolher um arquivo por vez.

 

2. Arrastar os arquivos desejados para a caixa Attributos:

                                                                                                                                                                  Figura 4: Ação Adicionar Atributos

 

Os formatos de arquivos suportados atualmente na interface são: .shp e .tif.

O número de arquivos adiconados a lista de atributos fica a escolha do usuário, e o processo pode ser repetido quantas vezes quiser.

 

As opções de inserção podem ser combinadas, ficam a critério do usuário utilizar a que mais lhe convém.

Após a inserção, o nome dos arquivos dos atributos aparecerá na lista:

 

                       Figura 5: Atributos Adicionados a Lista

Uma vez que um atributo é adicionado a lista, o usuário pode selecionar a operação desejada, para que esse dado fique disponível no espaço celular.

Para configurar uma operação, basta clicar no nome do arquivo presente na lista, e janela entrará no modo configuração de operação:

                                                       Figura 6: Janela Seleção de Operação

 

A lista de operações disponíveis é gerada de acordo com o formato do arquivos, estando as operações vetoriais para os arquivos .shp e as raster para os aquivos .tif.

Selecionada a operção desejada, os parâmetros de configuração da operação irão aparecer:

 

                                                          Figura 7: Janela Preenchendo Operação

 

Após salvar os dados da operação, a confirmação aparecerá na coluna Status:

               Figura 8: Confirmação Operação Configurada
