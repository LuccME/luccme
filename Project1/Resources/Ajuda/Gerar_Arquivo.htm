Como o nome sugere, esta sessão é responsável pela geração do arquivo (.lua).

Será salvo um arquivo, sendo o nome do script + a extensão .lua.

Certifique-se de preencher todos os dados necessários e clique no botão Gerar Arquivo:

                                                              Figura 1: Botão Gerar Arquivos

 

Caso a gravação ocorra com sucesso uma mensagem de confirmação será exibida:

 

                              

                                                  Figura 2: Arquivo gravado com sucesso

 

Após a geração do Arquivo será exibido o botão Rodar Script, possibilitando executar o script que acabou de ser salvo.

                                                              Figura 3: Botão Rodar Script
