Como o nome sugere, esta sessão é responsável pela geração dos arquivos (.lua).

Serão salvos dois arquivos: um sendo o nome do modelo com _main.lua, contendo as informações gerais do modelo e outro sendo o nome do modelo com _submodel.lua, contendo os submodelos  configurados.

Certifique-se de preencher todos os dados necessários e clique no botão Gerar Arquivos:

Caso a gravação ocorra com sucesso uma mensagem de confirmação será exibida:

          Figura 1: Botão Gerar Arquivos

 

Caso a gravação ocorra com sucesso uma mensagem de confirmação será exibida:

 

                                          

                                                       Figura 2: Arquivos gravados com sucesso

 

Após a geração dos Arquivos será exibido o botão Rodar Modelo, possibilitando executar o modelo que acabou de ser salvo.

 

                                                                Figura 3: Botão Rodar o Modelo
