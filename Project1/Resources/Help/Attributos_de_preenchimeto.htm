In this section the user must indicate which attributes will be used to fill the cells.

                                               Figure 1: Attributes to Fill

 

To add the files that contain the data, the user has two options:

1. Click on + Button, located at the bottom left of the Attributes to Fill Window:

                             Figure 2: Add Attributes Button

 

After clicking the button, the file selection window will appear:  

                                         Figure 3: Attribute Selection Window 

 

The file formats currently supported in the interface are: .shp and .tif.

The multi-select option is enabled in this window, so you do not have to choose one file at a time.

 

2. Drag the files you want to the Attributes Box:

                                                                                                                                           Figure 4: Add Attributes Action 

 

The file formats currently supported in the interface are: .shp and .tif.

The number of files added to the attribute list is a user's choice, and the process can be repeated as many times as desired.

 

The insert options can be combined, it is up to the user to use the one that suits him best.

After insertion, the name of the attribute files will appear in the list:

 

              Figure 5: Attributes Added to List 

Once an attribute is added to the list, the user can select the desired operation so that this data becomes available in the cell space.

To configure an operation, simply click on a name of the file in the list, and the window will enter configuration mode:

                                                  Figure 6: Operation Selection Window 

 

The list of available operations is generated according to the files format, with the vector operations for .shp files and raster operation for the .tif files.

When the desired operation is selected, the operation configuration parameters will appear:

 

                                                  Figure 7: Configuring Operation Window 

 

After saving the operation data, the confirmation will appear in the Status column:

    Figure 8: Configurated Operation Confirmation
