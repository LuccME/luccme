This section handles with the files generation (.lua).

A file will be saved named as the Script name + the extension .lua.

Be sure to fulfill all the necessary data to make the file.

In order to make the file click on Make File Button:

                                                                    Figure 1: Make File Button

 

After a success file generation a message will show:

                                

                                                        Figure 2: Files successfully recorded 

 

The Run Script Button will be visible after the file has been recorded, enabling to the user to execute the script that was saved.

                                                                   Figure 3: Run Script Button
