Coord

A spatial location, represented by a pair (x, y).

Parameters

  • x: A position on the horizontal axis of a two-dimensional Cartesian coordinate system. Defalt is 0.
  • y: A position on the horizontal axis of a two-dimensional Cartesian coordinate system. Defalt is 0.

Usage

coord = Coord()
coord2 = Coord{x = 2, y = 3}
print(coord2.x) -- nil
