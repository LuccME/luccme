Neighborhood

Each Cell has one or more Neighborhoods to represent proximity relations. A Neighborhood is a set of pairs (cell, weight), where cell is a neighbor Cell and weight is a number storing the relation's strength. This type is used to create neighborhoods from scratch to be used by Cell:addNeighborhood(). To create well established neighborhoods see CellularSpace:createNeighborhood(). Neighborhoods can also be loaded from external soures using CellularSpace:loadNeighborhood(). Calling forEachNeighbor() from a Cell traverses one of its Neighborhoods.

Parameters

  • id: a unique identifier for the neighborhood. As default, TerraME uses a string with an auto incremented number.

Usage

n = Neighborhood()

addNeighbor 

Add a new Cell to the Neighborhood. It returns a boolean value indicating whether the Cell was correctly added.

Parameters

  • 1st: A Cell to be added.
  • 2nd: A number representing the weight of the connection. Default is zero.

Usage
