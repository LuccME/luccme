Types

AllocationCClueLike Based on the process of competition among classes in the same cell, adjusted iteratively to reach the demand when all cells are considered, as described in Verburg et al.
AllocationCClueLikeSaturation Modification of the AllocationClueLike component.
AllocationDClueSLike Based on the process of competition among classes in the same cell, adjusted iteratively to reach the demand when all cells are considered, as described in Verburg et al.
AllocationDClueSNeighOrdering Based on the process of competition among classes in the same cell, adjusted iteratively to reach the demand when all cells are considered, as described in Verburg et al.
AllocationDSimpleOrdering Simple model developed as teaching material.
DemandComputeThreeDates A component (demand algorithm), where the demand inputs set by the user on the configuration file is used to distribute on the cells the correct among of this demand according to the parameters rules.
DemandComputeTwoDates A component (demand algorithm), where the demand inputs set by the user on the configuration file is used to distribute on the cells the correct among of this demand according to the parameters rules.
DemandExternalModel A component that call an external model to generate the demand.
DemandPreComputedValues A component (demand algorithm), where the demand inputs set by the user on the configuration file is used to distribute on the cells the correct among of this demand according to the parameters rules.
LuccMEModel Handles with a LuccME model behaviour.
PotentialCLinearRegression Uses linear regression techniques to estimate the cell potential for change.
PotentialCSampleBased Component developed based on MaxEnt behaviour, considering the attributes values of the samples cells to calculate the potencial of the use on the whole cellular space.
PotentialCSpatialLagLinearRegressionMix Modification of the SpatialLagRegression component to allow the use of simple linear regression in specific cases, when roads are created or paved.
PotentialCSpatialLagRegression Similar to the LinearRegression approach, but relies spatial regression techniques to estimate the regression cover (considers the spatial dependence of the land use).
PotentialDInverseDistanceRule Simple component developed as teaching material.
PotentialDLogisticRegression Uses logistic regression techniques to compute cell probability for each land use.
PotentialDLogisticRegressionNeighAttract Modification of the LogisticRegression combining cellular automata based models ideas.
PotentialDLogisticRegressionNeighAttractRepulsion Modification of the LogisticRegression combining cellular automata based models ideas.
PotentialDNeighInverseDistanceRule Simple component developed as teaching material.
PotentialDNeighSimpleRule Simple component developed as teaching material.
